Seminar Focus

기본 흐름에서 실제 제작 경험으로

1회 세미나에서 바이브코딩의 기본 흐름을 경험했다면, 2회에서는 실제로 쓸 수 있는 웹앱 제작에 집중합니다.

1회 참석자가 아니더라도 기본적인 바이브코딩을 직접 해본 경험이 있다면 참여할 수 있습니다.

취약 유형 파악

반복 학습 관리

Seminar Flow

세미나에서 다룰 내용

01

웹앱 아이디어 구체화

수업과 학원 업무에서 앱으로 만들 만한 문제를 선별합니다.

02

프롬프트 작성법

AI에게 목적, 기능, 화면, 데이터를 설명하는 요구사항을 정리합니다.

03

화면 구성 기획

오답노트 관리 앱의 목록, 입력, 상세, 상태 관리 화면을 설계합니다.

04

데이터 구조 설계

학생, 문제, 오답 유형, 복습 상태 같은 핵심 정보를 구조화합니다.

05

AI와 함께 구현·수정

웹앱을 만들고 오류를 수정하며 원하는 형태에 가깝게 개선합니다.

06

수업 현장 적용

완성된 결과물을 실제 수업과 상담 흐름에 연결하는 방법을 확인합니다.
