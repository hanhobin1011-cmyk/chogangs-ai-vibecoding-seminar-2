# AI 바이브코딩 세미나 2회 참석자 안내

## 한 줄 요약

이번 세미나는 수학강사가 AI와 함께 수업과 학원 업무에 필요한 간단한 웹앱을 직접 기획하고 구현해보는 실전 세미나입니다.

## 핵심 주제

- 바이브코딩 실전편 - 웹앱 만들기
- 실습 예시: 오답노트 관리 앱
- 목표: 코딩을 전문적으로 배우는 것이 아니라, 수업 현장의 문제를 웹앱 형태로 정리하고 AI와 함께 구현하는 경험을 해보는 것

## 참여 대상

- AI 바이브코딩 세미나 1회를 들은 선생님
- 1회 참석자는 아니지만 기본적인 바이브코딩을 직접 해본 경험이 있는 수학강사
- 수업자료, 오답관리, 학원 업무를 더 효율적으로 관리하고 싶은 선생님
- 코딩 실력보다 아이디어와 수업 현장 경험을 바탕으로 도구를 만들어보고 싶은 선생님

## 왜 오답노트 관리 앱인가?

오답 관리는 수학 수업에서 학생의 취약 유형, 반복 학습 상태, 상담 근거 자료를 정리하는 데 중요합니다. 세미나에서는 이 문제를 예시로 삼아 다음 흐름을 경험합니다.

- 학생별 취약 유형 정리
- 오답 유형과 단원 흐름 기록
- 복습 상태 관리
- 상담에 활용할 근거 자료 구조화
- AI에게 요구사항을 설명하고 웹앱으로 구현하기

## 세미나에서 다룰 내용

1. 웹앱 아이디어를 구체화하는 방법
2. AI에게 요구사항을 설명하는 프롬프트 작성법
3. 오답노트 관리 앱의 화면 구성 기획
4. 학생, 문제, 오답 유형, 복습 상태 같은 데이터 구조 설계
5. AI와 함께 웹앱을 만들고 수정하는 실전 흐름
6. 완성된 결과물을 수업 현장에 적용하는 방법

## 준비하면 좋은 것

- 본인이 관리하고 싶은 수업 또는 학원 업무 사례 1개
- 오답 관리에서 불편했던 점 1~2가지
- 바이브코딩을 시도해본 경험 또는 사용해본 AI 도구
- 세미나 중 함께 개선해보고 싶은 간단한 앱 아이디어

## 이 세미나가 아닌 것

- 개발자를 양성하는 과정이 아닙니다.
- 코딩 문법을 전문적으로 배우는 시간이 아닙니다.
- 완성도 높은 상용 서비스를 만드는 과정이 아닙니다.

## 이 세미나가 지향하는 것

수학강사가 자신의 현장 경험을 바탕으로 AI에게 필요한 일을 설명하고, 실제로 사용할 수 있는 작은 업무 도구를 만들어보는 경험입니다.
