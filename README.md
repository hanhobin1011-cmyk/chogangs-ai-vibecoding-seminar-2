# AI 바이브코딩 세미나 2회

초강스 수학강사 선생님들을 위한 내부 홍보 및 참석자 안내 자료입니다.

## 공개 페이지

- 세미나 안내 페이지: https://hanhobin1011-cmyk.github.io/chogangs-ai-vibecoding-seminar-2/
- 로컬 미리보기: `index.html`

## 세미나 개요

- 세미나명: AI 바이브코딩 세미나 2회
- 핵심 주제: 바이브코딩 실전편 - 웹앱 만들기
- 실습 예시: 오답노트 관리 앱 만들기
- 참여 대상: 1회 세미나를 들은 선생님, 또는 기본적인 바이브코딩을 직접 해본 경험이 있는 수학강사
- 목적: 개발자가 되는 것이 아니라, 수학강사가 자신의 수업과 학원 업무에 필요한 간단한 웹앱을 AI와 함께 직접 기획하고 구현해보는 실전 경험 제공
